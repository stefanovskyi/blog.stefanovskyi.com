# Tech Stack

## Core Technologies
- **Static Site Generator:** [Hugo](https://gohugo.io/) - Used for building the blog from Markdown source files.
- **Base Theme:** [Cactus](https://github.com/monkeyWzr/hugo-theme-cactus) - A clean, responsive theme for personal blogs.
- **Content Format:** [Markdown](https://www.markdownguide.org/) - For writing blog posts and pages.
- **Configuration & Metadata:** [TOML](https://toml.io/) - Used for the site configuration (`hugo.toml`) and post front matter.

## Frontend & Styling
- **Styling:** [SCSS/SASS](https://sass-lang.com/) - Used for custom styling and theme overrides (located in `assets/scss`).
- **Diagrams:** [Mermaid.js](https://mermaid.js.org/) - Integrated via CDN for rendering charts and diagrams from Markdown.
- **Icons:** [Font Awesome](https://fontawesome.com/) - Integrated for social icons and UI elements.
- **Typography:** [JetBrains Mono](https://www.jetbrains.com/lp/mono/) - Primary font used for code blocks and UI elements.
- **Scripts:** [jQuery](https://jquery.com/) - Included for theme-specific interactions and scripts.

## Hosting & Environment
- **Base URL:** `https://blog.stefanovskyi.com`
- **Deployment:** Static site deployment, hosted on GitHub Pages.
